Self-Service Tools

Welcome to tools. These are a collection of scripts that don't impact the server performance wise and are safe to run by admins.

Scripts

  • Print JVM Params
    Prints basic parameters about the JVM the instance is running on
  • Kicks egraphs awaiting verification
    Processes all egraphs that have been posted by a celeb but not verified by biometrics. Pushes the egraphs to biometrics, in order to get to the admin flow.
  • Kicks pending enrollment batches
    Enrolls all celebrities that are stuck awaiting enrollment.
